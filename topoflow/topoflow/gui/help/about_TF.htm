About TopoFlow

TopoFlow Version 1.5, Build date: 12/18/07

Copyright (c) 2002-2007.

TopoFlow is the result of contributions from multiple authors and partial financial support from the University of Alaska at Fairbanks. This next-generation hydrologic model combines proven strategies from several other models including ARHYTHM and an advanced, D8-based rainfall-runoff model. It also has a user-friendly graphical interface, an extensible architecture and an extensive set of supporting tools. Each author retains the copyright to their own contributions. Please see the copyright statements at the start of each source code file.

Contributors to the TopoFlow project include:

Dr. Scott Peckham Overall design, oversight, most source code, testing, documentation and the website.
Dr. Larry Hinzman ARHYTHM model author, funding support and ongoing collaboration.
Dr. Matt Nolan Collaboration on user interface and design.
Dr. Bob Bolton Variable hydraulic conductivity method for subsurface flow and the Inverse Distance Method routine used by the "RTS File from Station Data" dialog in the Create menu.
Dr. Thomas Over Multifractal cascade routines for simulated space-time rainfall.
Anna Liljedahl Testing and detailed bug reports.

More information regarding the TopoFlow project can be found at: http://instaar.colorado.edu/topoflow/ The authors welcome new contributors and feedback from users.
