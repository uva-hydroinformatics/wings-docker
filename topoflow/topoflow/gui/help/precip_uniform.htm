Precipitation → Uniform Method

These are the input variables used for the Precipitation method.

rate = precipitation rate [mm/hr]
duration = precip. duration of a given rate [minutes]
T_air = air temperature [deg C]

If air temperature is specified for another physical process then precipitation falls as snow (and snow depth is incremented) whenever the air temperature drops below 0 degrees Celsius.

Rainrates are read and written from files with units of [mm/hr]. Durations are read and written from files with units of [min].
